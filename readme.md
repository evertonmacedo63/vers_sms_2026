# VERS SMS 2026

Repositório da disciplina VERS SMS 2026 para acompanhamento dos trabalhos dos alunos.

## Objetivo deste repositório

- Centralizar os links dos repositórios dos alunos
- Registrar orientações de atividades
- Servir como modelo de organização para os alunos

## Como os alunos devem usar este modelo

Cada aluno deverá:
- Criar seu próprio repositório no GitHub
- Adicionar um README.md baseado no modelo sugerido pelo professor
- Compartilhar o link do repositório aqui (em uma lista que será organizada pelo professor)
